# Python Programming: Learning WITH AI
## A 12-Week Course Using "Python Step by Step: Learning with AI"

### Course Philosophy - REVISED
"AI is your learning partner and code assistant. Master the concepts, let AI handle the syntax, but always understand what you're building."

### Core Principles
1. **Computational Thinking First**: Understand the problem before the code
2. **AI as Teacher**: Use AI to explain, trace, and understand
3. **Progressive Prompting**: Learn to guide AI from simple to complex
4. **Architecture Over Syntax**: Be the designer, not just the coder
5. **Verification Skills**: Always validate AI's output

### Required Textbook
**"Python Step by Step: Learning with AI"** - Adapted for AI-integrated learning

## Standard Lecture Structure

Every week follows this pattern:

1. **Hook/Reality Check** (5-10 min)
   - Real-world problem or AI demonstration
   - Why this week's topic matters

2. **Computational Concepts** (15-20 min)
   - Abstract concepts first
   - Real-world analogies

3. **From Concepts to Code** (10-15 min)
   - Bridge abstract to concrete
   - Map concepts to Python syntax
   - Introduce new features explicitly

4. **AI Partnership Demo** (10-15 min)
   - Progressive prompt refinement
   - Constraints based on cumulative knowledge

5. **Integration & Application** (5-10 min)
   - Why it matters
   - Practical uses

## Key Differences from Traditional Approach

1. **Embrace AI**: Use it strategically, not avoid it
2. **Concepts First**: Understand why before how
3. **Prompt Engineering**: Core skill throughout
4. **Mental Models**: Visualize before coding
5. **Architecture Focus**: Design systems, not just syntax
6. **No Surprises**: Every feature explicitly introduced before use

## Success Metrics

- Students can architect solutions and guide AI implementation
- Students can debug AI-generated code effectively
- Students understand when and why to use programming concepts
- Students can adapt to new problems using learned patterns
- Students are prepared for real-world development with AI tools

## The Honest Promise

"By the end of this course, you won't memorize every Python syntax detail - AI can help with that. Instead, you'll understand how to think computationally, design solutions, and guide AI to build what you envision. You'll be a programmer who uses AI as a powerful tool, not a crutch."
