# Week 1: Python Foundations & AI Partnership

## Overview
Week 1 introduces students to Python programming WITH AI assistance using Google Colab (zero-install environment).

## Structure
- **Lecture**: 1 hour (covers all foundational concepts)
- **Lab**: 2 hours (hands-on practice and project work)

## Key Topics
1. Course philosophy: AI as learning partner and code assistant
2. Computational thinking: 6 building blocks (Input, Storage, Processing, Output, Decisions, Repetition)
3. From concepts to code: Python data types (str, int, float, bool)
4. Critical concept: Type conversion (why input() returns strings)
5. Progressive prompt engineering with AI
6. Building mental models before coding

## Materials
- `week-01-crd.md` - Course Requirements Document
- `week-01-task-list.md` - Development task list
- `lectures/` - Lecture content and slides

## Learning Outcomes
By end of Week 1, students can:
- Use Google Colab effectively
- Write basic Python programs
- Debug simple errors
- Use AI appropriately for learning
- Understand fundamental data types
- Create interactive programs

## Project
Temperature Converter - A simple but complete program demonstrating all Week 1 concepts.
