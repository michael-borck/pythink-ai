# Python Programming: Learning WITH AI
## A 12-Week Course Using "Python Step by Step: Learning with AI"

### Course Philosophy
"AI is your tutor, not your ghostwriter. Use it to understand, not to skip understanding."

### Required Textbook
**"Python Step by Step: Learning with AI"** - The course follows this book chapter by chapter

### Pre-Course Setup
Students create an "AI Learning Profile":
1. Set up Python environment (Book Appendix A)
2. Choose and configure AI assistant (Book Appendix B)
3. Create course folder structure
4. Read Chapter 0: "Your AI Learning Partner"

## Week 1: First Steps with Python and AI

### Learning Objectives
- Understand how AI can help (and hinder) learning
- Master print(), variables, and input()
- Learn to simplify AI's complex solutions
- Establish good learning habits with AI

**Lecture (1hr):**
- Course introduction and expectations
- Demo: "Make a calculator with AI" (watch it fail when modified)

- The Simplification Protocol

**Lab (2hr):**

  - Try complex project with AI
  - Discover modification problems

- **Hour 2: Chapter 1 Exercises**
  - Fix AI's broken print statements
  - Simplify AI's complex outputs
  - Trace every character

### Wednesday: Variables and Input

**Lecture (1hr):**

- Getting user input

- Tracing values step-by-step

**Lab (2hr):**

- **Special Focus**: AI gives you complex input validation - strip it away
- Partner activity: Explain variables to each other
- Build simple conversation program

### Friday: Numbers and Integration
**Reading**: Chapter 4
**Lecture (1hr):**
