## Key Constraints for Week 2

### Must Use ONLY
From Week 1:
- print(), input()
- Variables
- Data types: str, int, float, bool
- Type conversion: int(), float(), str()
- Basic math: +, -, *, /
- String concatenation
- f-strings
- .upper(), .lower()
- Comments

New in Week 2:
- if, elif, else
- Comparison: ==, !=, <, >, <=, >=
- Boolean: and, or, not
- Indentation for blocks
- True, False values

### Must NOT Use
- Loops (for, while)
- Lists
- Functions (def)
- Dictionaries
- try/except
- imports
- match/case
- Ternary operators
- List comprehensions

- Students understand decision flow

- Write working conditional code
- Debug logic errors effectively
- Use AI appropriately for learning
- Complete grade calculator project
