# Week 2: Decision Making & Program Flow

## Overview
Week 2 introduces conditional logic, teaching students how programs make decisions. Building on Week 1's foundations, students learn to create programs that respond intelligently to different inputs.

## Learning Objectives
- Understand and apply if/elif/else statements
- Use comparison operators (==, !=, <, >, <=, >=)
- Combine conditions with logical operators (and, or, not)
- Create flowcharts to visualize program logic
- Debug logic errors systematically

## Structure
- **Lecture**: 1 hour covering decision concepts and Python implementation
- **Lab**: 2 hours of hands-on practice with conditionals
- **Project**: Smart Grade Calculator with full grade scale

## Key Concepts
1. **Computational Thinking**: How programs make choices
2. **Three Decision Patterns**: if, if/else, if/elif/else
3. **From Concepts to Code**: Mapping decisions to Python syntax
4. **Common Pitfalls**: Order matters, indentation critical, = vs ==

## Prerequisites
Students must have completed Week 1 and understand:
- Variables and data types
- Input/output operations
- Type conversion (int, float, str)
- Basic AI partnership approach

## Common Challenges
- **Indentation**: Python's unique use of whitespace
- **Logic Order**: elif chains must be ordered correctly
- **Comparison Operators**: Distinguishing = from ==
- **Edge Cases**: Testing boundary values

## Looking Ahead
Week 3 will introduce loops, which use these same conditional concepts to repeat actions based on conditions.
