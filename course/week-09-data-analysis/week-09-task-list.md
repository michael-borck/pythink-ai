## Key Constraints for Week 9

### Must Use ONLY
From Weeks 1-8:
- All previous features (variables, loops, conditions, functions, lists, strings, dictionaries, files, organization)

New in Week 9:
- CSV file parsing (manual)
- Statistical calculations (manual implementation)
- Text-based visualizations
- Data cleaning strategies
- Analysis pipelines

### Must NOT Use
- pandas library - Beyond course scope
- numpy - Beyond course scope
- matplotlib/seaborn - Beyond course scope
- csv module - We implement manually
- statistics module - We implement manually
- External visualization libraries

### Google Colab Considerations
- File upload methods
- Output formatting
- Text-based charts only
- Memory limitations for large files
